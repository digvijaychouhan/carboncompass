# 🌍 CarbonCompass

A daily-updated knowledge hub for carbon and climate compliance — built to track news, regulations, and frameworks relevant to carbon accounting and climate risk.

🔗 **Visit the site:** [digvijaychouhan.github.io/carboncompass](https://digvijaychouhan.github.io/carboncompass)

## What's inside
- **Daily news feed** — auto-pulled climate/carbon regulatory and market news
- **Framework guides** — CBAM, GHG Protocol, ISO 14064
- **Startup tracker** — companies working in carbon accounting/credits
- **Certifications reference** — relevant carbon/climate credentials

## How it's built
A lightweight, self-updating site — Python script pulls news daily via GitHub Actions, no manual maintenance, zero hosting cost (GitHub Pages).

## Status
Early-stage reference project. Content is being reviewed for accuracy — treat as a starting point, not a verified compliance source, until noted otherwise.

## About
Built by [Digvijay Chouhan](https://linkedin.com/in/digvijaychouhan), GRC & Information Security leader exploring automation applied to compliance domains beyond information security.
